# Unity 게임개발 심화 팀프로젝트

<br/>

## Project Introduction

3D 기반 턴제 RPG 게임으로 재료 아이템을 모아서 아이템을 조합하고 

전투를 턴제 시스템으로 진행합니다. 슬롯을 돌려서 선택한 행동의 효과가 정해지는것이 특징입니다.

Team Notion : https://teamsparta.notion.site/777-e7aa1783f7e54747b9a3ce2455fe4992
